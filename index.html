Exercise 3

This exercise requires you all to work collaboratively to make your own individual changes, create pull requests and merge all of your changes.

Task 1:

You will each create a new branch and write a short decription of yourself in the 'Meet The Team Section'.

Once completed, you will commit and push your change to the branch you created.

Once you've pushed your changes, go to the GitHub repository and open a Pull Request.

The person responsible for managing the repository will review the pull request and approve it if everything looks good.

After approval, they will merge the pull request into the main branch.

Task 2:

Insert some text in the 'contact' page on the same line as someone else and try to merge it to see what happens.

This is where we'll be learning to deal with merge conflicts.
